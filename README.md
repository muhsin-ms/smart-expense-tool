# 💸 Smart Expense Prediction Tool

An AI-inspired web application that helps users estimate and manage their expenses efficiently.

## 🚀 Live Demo
(https://smart-expense-tool.vercel.app/)

---

## 📌 Features

- 📊 Calculates monthly and yearly expenses
- 💰 Compares expenses with income
- 🎯 Tracks savings goals
- ⚠️ Smart spending insights and suggestions
- 🧾 Category-based expense analysis (Food, Travel, Shopping)
- 📱 Responsive and clean UI

---

## 🧠 How It Works

1. Enter your daily expense  
2. Enter your monthly income  
3. Set your savings goal  
4. Add category expenses  

👉 The app will:
- Calculate your total spending  
- Show percentage of income spent  
- Give smart financial suggestions  

---

## 🛠️ Tech Stack

- HTML  
- CSS  
- JavaScript

## ⚡ Example Output
